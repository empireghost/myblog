### oracle 判断某个字段是否是数值

```java
SELECT distinct con.ACCOUNT_BANK_NAME
  FROM USER_BANK CON
 WHERE TRIM(TRANSLATE(CON.ACCOUNT_BANK_NAME, '0123456789', ' ')) IS NULL;

```





```sql
SELECT DISTINCT CON.ACCOUNT_BANK_NAME
  FROM USER_BANK CON
 WHERE CON.CDEFAULT_FLAG = '0'
   AND TRIM(TRANSLATE(NVL(CON.ACCOUNT_BANK_NAME, 'X'), '0123456789', ' ')) IS NULL;
```
