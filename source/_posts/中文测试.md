###### *oracle 排序*
```
    SELECT
        CONTRACT_ID AS contractID,
        OPE_TIME AS opeTime
    FROM
        (
            SELECT
                ROW_NUMBER () OVER (
                    PARTITION BY IRB.CONTRACT_ID
                    ORDER BY
                        IRB.OPE_TIME DESC
                ) AS rn,
                IRB.*
            FROM
                ICP_REPAYMENT_BOOKING IRB
            WHERE
                IRB.STATE_CODE IS NOT NULL
        )
    WHERE
        RN = 1
```
